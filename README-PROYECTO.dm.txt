# Descripción del Proyecto

En este proyecto se ha creado desde cero mi propia página web portfolio. Incluye una breve descripción de mi trayectoria profesional, muestra diversos proyectos tanto de desarrollo web (ficticios) como de diseño gráfico y, finalmente, facilita medios de contacto. Para desarrollar la página, se realizaron una serie de esbozos en papel de la estructura/esqueleto de la web, y una vez decidido esto, se creó el mockup final en Figma. Posteriormente, se procedió a la fase de desarrollo, utilizando tecnologías HTML5 y CSS3.

## Instalación y Uso

Para instalar este proyecto, sigue estos pasos:

1. Clona este repositorio en tu ordenador local.
2. Navega hasta el directorio del proyecto.
3. Abre el archivo `index.html` en tu navegador web preferido.
